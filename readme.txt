== Description ==
Login plus changes Wordpress Login Logo and Logo Url with out altering any core file. See Login Logs like Hacking attempt,Failed login, successfull login with tons of info.



== Installation ==

Install Login Plus automatically through the WordPress Dashboard or by uploading the ZIP file in the plugins directory. Then, after the package is uploaded and extracted, click Activate Plugin.
After Activating plugin you can see a LoginPlus menu in Wordpress Dashboard Sidebar.
